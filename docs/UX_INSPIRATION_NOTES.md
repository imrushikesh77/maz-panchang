# UX Inspiration Notes (Play Store Study)

Reference search used:

- Marathi calendar app listings on Google Play
- Example app: Kalnirnay listing (`com.awt.kalnirnay`)

Goal was inspiration only (not copying design assets/text).

## Common patterns seen in top apps

1. Printed-calendar familiarity (month grid first)
2. Fast daily detail from date selection
3. Category-based event browsing (festival, holiday, fasting)
4. Reminder controls for recurring events
5. Smooth month navigation and low-latency rendering

## What was implemented in this app (Phase-3)

- Warm traditional color palette and icon-led tab bar
- Large Marathi date hero on the `आज` screen
- Swipe navigation for previous/next day
- Square month grid with printed-calendar feel
- Date tap opens full day detail sheet
- Weekend visual cues (Saturday/Sunday colors)
- Per-cell mini tithi text
- Event markers for festival/upwas
- Month prewarm cache for adjacent months
- District-aware jatra rule mapping
