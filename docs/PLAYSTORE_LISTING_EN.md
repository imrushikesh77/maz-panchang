## Full Description

Maza Panchang (माझं पंचांग) is a Marathi-first calendar app made for Maharashtra users, especially for simple and fast daily use on budget Android phones.

What you get every day:

- Tithi (तिथी), Vara (वार), Nakshatra (नक्षत्र), Yoga (योग), Karana (करण)
- Sunrise and sunset times
- Rahukal, Gulikkal, Yamagand, Abhijit Muhurat
- Festivals, vrat, and upwas highlights

Key benefits:

- Full Marathi UI
- District-based Maharashtra calculation
- Big text mode for easy reading
- Low-memory friendly performance
- Daily reminder at 7:00 AM
- Tomorrow special alert at 7:00 PM (if festival/upwas)

Built with a practical rural-first design:

- Clean layout
- Low confusion navigation
- Fast loading and minimal battery impact

Important note:

- This app is independently developed. It is not affiliated with any other calendar brand.
